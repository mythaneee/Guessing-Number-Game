# Number Guessing Game

A simple, fun and interactive console-based game written in Python.

The computer generates and randomly selects a number between 1 and 100.
A limited number of attempts, up to 10 is given.
After each game, points are awarded according to how fast you guess the number.

## Features
Random number generator (1-100)
Limit of 10 attempts.
Prompted hints: "Too high!/Too Low!"
Validated inputs (Only Integers count)
Scoring system and points awarded based off of attempts
Stores and reveals your guesses made in every attempt.

Scoring system:

- ≤ 3 attempts → 5 points

- ≤ 7 attempts → 3 points

- ≥ 7 attempts → 1 point

-   10 attempts = 0 point

## How to Play
Run the Game
Guess the number when prompted
Follow given hints: "Too High! / Too Low!"
Win by guessing the number or use all 10 attempts → see your guesses
Choice of whether to play again
Decide to quit, and see your games played and total score.
